# House Price prediction using Machine Learning - Simple ML Project

## Overview 
This project demonstrates a simple machine learning model that predicts house prices based on house size using linear regression.

This project demonstrates how regression models can capture relationships in structured data and form the basis for more complex predictive modelling systems.

## Objective
To understand how machine learning learns patterns from data and makes predictions.

## What I learned
- Data preprocessing
- Train-test split
- Linear regression model
- Model evaluation (R² score, MSE)
- Data visualization

## Tech Stack
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Result
The model predicts house prices based on size and shows a clear linear relationship between variables.
